DataManipulation
================
It performs various operations on data mostly contained in List<String>, along with some other file operations. 

In specific it performs that following: 

 * clearListString: it removes the null and the duplicate items from a List<String> 
 * AppendWordList: it appends a List<String>, to a file provided as an input in form of a String
 * AppendString: it appends a String to a file provided as an input in form of a String
 * AddAList: it appends a List<String> to another List<String>
 * removeChars: it removes useless characters from a String
 * sortHashMap : it sorts a HashMap
 * getinputfiles: it creates a collection of files from a certain directory and for a certain filetypes
